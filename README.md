# 프로젝트 이름

nodejs-practice

## 소개

nodejs 를 이용한 서버 구축 연습.

### Dependencies
* nodejs
* html
* css
* mysql

### 프로그램 실행 방법 
```
cd 원하는 폴더 
npm i 
npm i -D
npx sequelize db:create
npm start
```
